# Projeto de Testes Manuais – Página de Login

# Objetivo
Validar o funcionamento da página de login através da criação de cenários e casos de teste manuais, contemplando cenários positivos e negativos.

# Escopo do Projeto
Foram elaborados casos de teste para:
- Login com dados válidos
- Login com senha inválida
- Campos obrigatórios em branco

# Documentação
A documentação completa dos casos de teste está disponível no arquivo PDF anexado neste repositório.

# Ferramentas Utilizadas
- Documentação manual
- GitHub para versionamento

---

Projeto desenvolvido como prática de estudos em QA (Quality Assurance).
